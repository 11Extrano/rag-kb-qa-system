# 前端（client）

RAG 知识库问答系统前端，技术栈：**React 18 + TypeScript + Rsbuild + Ant Design**。

- **文档管理页**：`/admin/documents`，管理员上传、列表、删除文档。
- **问答页**：`/` 或 `/qa`，用户输入问题、展示答案与引用。

开发时 API 请求需指向后端地址（默认 `http://127.0.0.1:7001`），见 `openspec/changes/setup-rag-kb-qa-system/product-pages.md`。

首版前端实现任务见仓库根目录 `openspec/changes/setup-rag-kb-qa-system/tasks.md` 第 7 节。
